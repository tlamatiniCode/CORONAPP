# CORONAPP_esp

## Web app, para ayudar ante el Covid-19.
En el estado actual la solución es estar en casa para prevenir.Pero hay personas que necesitan salir de sus hogares, ya sea por trabajo, viajes u otros temas.Por esa razón he realizado esta web app que permite priorizar los test del covid a aquellas personas cuyas necesidades sean más prioritarias que las de otras personas que puedan quedarse en casa.

## Como funciona?
La idea es que con un ID, respectivo en cada lugar o común para todos, lo insertes y solicitar dependiendo de tu prioridad para hacerte el test.Una vez realizado el test el doctor puede cambiar tu estado, de este modo, enseñado tu estado o diciendo tu ID pueden verificar si has realizado el test y en que estatus estas, es decir, curado, si ha dado positivo o negativo.

## A partir de aquí que sigue?
Esta es una demo, de lo que podría ser de utilidad para los ciudadanos y el estado, gobierno, ya que si ese ID esta asociado a una dirección se puede mapear donde estarían los focos de contagio.Entonces una de las posibles actualizaciones sería con toda la información recabada hacer un mapa.Los datos que se utilizan en esta demo son estáticos y de ejemplo, faltaría unir una BBDD donde la info del ID y la de los tests estuvieran unidas.

Im gegenwärtigen Zustand besteht die Lösung darin, zu Hause zu sein, um dies zu verhindern. Es gibt jedoch Menschen, die ihr Zuhause verlassen müssen, sei es wegen Arbeit, Reisen oder anderen Problemen. Aus diesem Grund habe ich diese Web-App entwickelt, mit der die Covid-Tests diesen vorrangig behandelt werden können Menschen, deren Bedürfnisse eine höhere Priorität haben als die anderer, die möglicherweise zu Hause bleiben.

## Wie funktioniert es?
Die Idee ist, dass Sie mit einer ID, die an jedem Ort vorhanden oder für alle gemeinsam ist, diese einfügen und je nach Priorität die Durchführung des Tests anfordern. Sobald der Test abgeschlossen ist, kann der Arzt Ihren Status ändern und so Ihren Status anzeigen oder Wenn Sie Ihre ID angeben, können sie überprüfen, ob Sie den Test durchgeführt haben und in welchem ​​Status Sie sich befinden, dh geheilt sind, wenn er positiv oder negativ ist.

## Was folgt von hier aus?
Dies ist eine Demo dessen, was für die Bürger und den Staat und die Regierung nützlich sein könnte, da diese ID, wenn sie mit einer Adresse verknüpft ist, dort abgebildet werden kann, wo sich die Ansteckungsherde befinden würden, sodass eine der möglichen Aktualisierungen bei allen liegen würde Die gesammelten Informationen bilden eine Karte. Die in dieser Demo verwendeten Daten sind statisch und es wäre beispielsweise erforderlich, einer Datenbank beizutreten, in der die ID und die Testinformationen verknüpft waren.
